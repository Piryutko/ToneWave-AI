# ToneWave AI – Roadmap & Future Improvements

Этот план развития (Roadmap) описывает возможные пути улучшения и расширения проекта ToneWave AI. Вы можете выбрать любую задачу из списка для дальнейшей реализации!

## 📌 Phase 1: Улучшение пользовательского опыта (UX/UI)
- [ ] **История запросов:** Сохранение истории транформаций в Local Storage (чтобы пользователь мог посмотреть, что он переводил вчера).
- [ ] **Кнопка "Очистить":** Быстрая очистка полей ввода и вывода.
- [ ] **Тост-уведомления (Toasts):** Красивые всплывающие сообщения об успешном копировании текста или возникших ошибках, вместо системных алертов.
- [ ] **Поддержка Markdown:** Если Ollama возвращает списки или жирный текст, отрендерить их на Svelte красиво (например, через `marked` или `mdsvex`).
- [ ] **Темная/Светлая тема:** Добавить переключатель тем (хотя текущий "стеклянный" темный дизайн выглядит очень круто).

## 🚀 Phase 2: Расширение функционала AI (ToneWave.Api)
- [ ] **Кастомные промпты:** Позволить пользователю вводить свои инструкции (например, "Переведи на английский вежливо" или "Сделай текст в стиле Шекспира").
- [ ] **Выбор моделей:** Если у вас скачано несколько моделей (llama3.2, mistral, qwen), добавить выпадающий список на фронтенд для переключения между ними "на лету".
- [ ] **Регулировка "Температуры":** Ползунок на фронтенде (Креативность vs Точность), который передается в API.
- [ ] **Анализ тональности:** Перед трансформацией выводить, насколько "токсичным" был оригинальный текст (по шкале от 1 до 10).

## 🛠 Phase 3: Архитектура, Стабильность и Деплой
- [ ] **Rate Limiting:** Добавить ограничение количества запросов к бекенду (чтобы кто-то не "положил" ваш домашний комп, если вы расшарите ссылку).
- [ ] **Централизованное логирование:** Использовать Serilog для записи логов в файл, чтобы было проще отлаживать ошибки.
- [ ] **Docker:** Упаковать бекенд `ToneWave.Api` в Docker-контейнер для более простого запуска на любых серверах (VPS).
- [ ] **Автоматизация Cloudflare:** Настроить Cloudflare Tunnel как системную службу (service), чтобы не нужно было каждый раз вручную держать терминал открытым.

## 📱 Phase 4: Новые платформы
- [ ] **Telegram Бот:** Сделать бота, которому можно переслать гневное сообщение, а он в ответ пришлет вежливый вариант (используя всё тот же `ToneWave.Api`).
- [ ] **Браузерное расширение (Chrome/Edge):** Выделяешь текст на любом сайте, нажимаешь правую кнопку мыши -> "Улучшить тон".
