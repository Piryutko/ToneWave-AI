# ToneWave AI Engine 🌊

> *Ваш личный дипломат: движок для трансформации текста*

**ToneWave AI Engine** — это ядро и backend-инфраструктура умного ИИ-переводчика с "эмоционального" на "конструктивный". 
Движок предназначен для работы в связке с веб-клиентом и использует локальные нейросети (через [Ollama](https://ollama.com/)) для преобразования резких или нецензурных высказываний в вежливый текстовый формат, полностью сохраняя изначальный смысл вашей мысли. 

## 🚀 Возможности ядра

Движок поддерживает 4 режима трансформации текста:
- 👔 **PROFESSIONAL**: Деловой стиль и конструктив. Идеально для бизнес-переписки.
- 💖 **KIND**: Максимальная эмпатия и поддержка. Превращает претензию в добрую и мягкую просьбу.
- ⚙️ **TECHNICAL**: Строгая и сухая аналитика. Удаляет воду и эмоции, оставляя чистые факты.
- ⚡ **LACONIC**: Сжатие до одной мощной и емкой фразы (5-7 слов) с сохранением главного посыла.

## 🛠 Технологии

- **C# 12 / .NET 8.0**
- Интеграция с HTTP REST API локальной или удаленной **Ollama** 
- Локальная языковая модель `llama3.2`

## 📦 Установка и запуск

### 1. Подготовка Ollama
Вам потребуется запущенный инстанс [Ollama](https://ollama.com/) с загруженной моделью `llama3.2`. 
По умолчанию ядро ищет Ollama на стандартном порту: `http://localhost:11434`.

Для загрузки и запуска модели выполните:
```bash
ollama run llama3.2
```

### 2. Запуск движка
Перейдите в папку с проектом и запустите его через .NET CLI:

```bash
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
cd ToneWaveCLI
dotnet run
```
В текущей версии (до завершения миграции в полноценный Web API) для демонстрации работы движка доступен тестовый CLI-интерфейс.
